# AWEMA 2030 Documentation

Welcome to the AWEMA 2030 documentation. AWEMA (Advanced Web Experience Management Architecture) is a modern monorepo-based web development platform.

## Getting Started

To get started with AWEMA 2030, check out our [Quick Start Guide](/getting-started).

## Architecture

AWEMA 2030 is built as a Turborepo monorepo with the following structure:

- **Apps**: Main applications (Studio, Preview Server, Docs)
- **Packages**: Core functionality, templates, editor, AI features, shared utilities, and analytics
- **Tools**: Build, deploy, and test utilities

## Features

- AI-powered website generation
- Visual editor with real-time preview
- Template system with customizable components
- Analytics and performance monitoring
- Deployment automation
