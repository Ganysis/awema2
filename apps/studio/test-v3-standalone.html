Problème Actuel

Erreur de Permissions

Le dossier .next appartient à root au lieu de l'utilisateur actuel, ce qui empêche le démarrage du serveur.

Solutions Possibles

  1. Solution 1 - Changer les permissions (recommandé) 
    sudo chown -R $USER:$USER .next
  2. Solution 2 - Supprimer et recréer 
    sudo rm -rf .next && npm run dev
  3. Solution 3 - Utiliser un port différent 
    PORT=3001 npm run dev
